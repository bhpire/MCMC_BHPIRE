double  vCo,
int  Nparam,

Calculates the model predictions.

Given a number of parameters Nparam and their values stored in the array Aparam[], it returns model prediction at the location with coordinates (uCo, vCo).

The model used here generates the visibility amplitude of a 2-Gaussian component model
